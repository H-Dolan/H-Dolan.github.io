EMS Incident Dispatch Visualization

The project aimed to analyze the EMS dispatch data and view the number of incidents and response time in Manhattan between 2013 and 2019.

Methodology: The dataset used was provided by the Fire Department of New York City. The dataset was broken up into multiple files in order to break down the file size. Static graphs were created using Tableau and dynamic graphs were created using JavaScrip.

Project created by: Hannah Dolan, Nathan Raymond, Emily Smith
