Hospital Incident Analysis

The objective of this project was to act as an incident book and analyze the number of and type of work-related accident using fictitious data.

Methodology: I gathered the data in an excel spreadsheet, utilizing two sheets for staff and a record of incidents. I created a data dictionary to categorize and define terms and incidents. Once finished, the files were exported as a CSV and imported them into a database. I queried the data required and used Tableau to visualize the results.
